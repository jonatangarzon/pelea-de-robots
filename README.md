# pelea-de-robots
una pelea de solo robots que se quieren destruir y dominar el mundo entre amos :)


Crea un programa en Python, usando POO, que simule el siguiente juego:

Crear una clase base llamada Robot con los atributos nombre, bateria y escudo.
Cree dos subclases: RobotAtaque y RobotDefensa.
El RobotAtaque debe tener un metodo atacar(objetivo) que reste vida a otro robot.
El RobotDefensa debe tener un método recargar() que suba sus puntos de escudo.
